## 🚀 Quick Start (GitHub Pages Deployment)

### Step 1: Create Your Repository
1. Go to [GitHub](https://github.com) and create a new repository
2. Name it `your-portfolio` (or any name you prefer)
3. Make it **Public** (required for free GitHub Pages hosting)

### Step 2: Upload Your Files
Upload these files to your repository:
```
📁 your-repo/
├── 📄 index.html          ← Main website file
├── 🖼️ 17581.png           ← Profile image
├── 🖼️ 17541.png           ← PowerPoint showcase 1
├── 🖼️ 17542.png           ← PowerPoint showcase 2
├── 🖼️ 17532.png           ← PowerPoint showcase 3
├── 🖼️ 17533.png           ← PowerPoint showcase 4
├── 🖼️ 17550.png           ← Video thumbnail 1
├── 🖼️ 17546.jpg           ← Video thumbnail 2
├── 🖼️ 17547.jpg           ← Video thumbnail 3
├── 🖼️ 17549.jpg           ← Video thumbnail 4
├── 🖼️ 17540.png           ← YouTube thumbnail 1
├── 🖼️ 17539.png           ← YouTube thumbnail 2
├── 🖼️ 17536.png           ← Graphic design 1
├── 🖼️ 17537.png           ← Graphic design 2
└── 🖼️ 17538.png           ← Graphic design 3
```

### Step 3: Enable GitHub Pages
1. In your repository, go to **Settings** (top right)
2. Scroll down to **Pages** section (left sidebar)
3. Under **Source**, select **Deploy from a branch**
4. Choose **main** branch and **/(root)** folder
5. Click **Save**
6. Wait 1-2 minutes, then visit your URL!

### Step 4: Your URL Will Be:
```
https://yourusername.github.io/your-repo-name
```

## 📝 Customization Guide

### Change the Name/Title
Edit the `<title>` tag and hero section in `index.html`:
```html
<title>Your Name — Creative Portfolio</title>
<h1 class="hero-title">YOUR NAME</h1>
```

### Update Contact Information
Find this section in `index.html` and update:
```html
<span>lamonjesury2004@gmail.com</span>  ← Your email
<span>+63 976 167 1480</span>           ← Your phone
```

### Add Your Social Media Links
Replace the `#` placeholders:
```html
<a href="https://facebook.com/yourprofile" title="Facebook">
<a href="https://instagram.com/yourprofile" title="Instagram">
<a href="https://tiktok.com/@yourprofile" title="TikTok">
<a href="https://youtube.com/yourchannel" title="YouTube">
```

### Replace Images
Simply replace the image files in your repository with new ones using the **same filenames**, or update the `src` attributes in the HTML.

## 🌟 SEO Tips

To improve search engine visibility:

1. **Add meta description** in the `<head>`:
```html
<meta name="description" content="Urydessy La Monjes — Virtual Assistant, Content Creator, and Designer specializing in video editing, Canva design, and social media management.">
```

2. **Add Open Graph tags** for social sharing:
```html
<meta property="og:title" content="Urydessy La Monjes — Creative Portfolio">
<meta property="og:description" content="Virtual Assistant • Content Creator • Designer">
<meta property="og:image" content="https://yourusername.github.io/your-repo/17581.png">
```

3. **Create a custom domain** (optional):
   - Buy a domain from Namecheap, GoDaddy, etc.
   - Add it in GitHub Pages settings under "Custom domain"
   - Create a `CNAME` file with your domain name
